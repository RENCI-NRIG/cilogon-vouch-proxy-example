CILogon / Vouch-Proxy Example

Choose from the examples below:

Flask (insecure) - http://127.0.0.1:8080/app/flask
Flask (secure) - https://127.0.0.1:8443/app/flask
Flask (curl) - set local variable COOKIE equal to the "Cookie" attribute from Flask (secure) above

$ curl --insecure --cookie $COOKIE https://127.0.0.1:8443/app/flask

React (insecure) - http://127.0.0.1:8080/app/react
React (secure) - https://127.0.0.1:8443/app/react
